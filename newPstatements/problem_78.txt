Given two integers n and k, return all possible combinations of k numbers out of the range [1, n]. You may return the answer in any order. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 4, k = 2 <br><h4>Output:</h4> [ [2,4], [3,4], [2,3], [1,2], [1,3], [1,4], ] <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 1, k = 1 <br><h4>Output:</h4> [[1]] <br><h4>Constraints:</h4> 1 <= n <= 20 1 <= k <= n
