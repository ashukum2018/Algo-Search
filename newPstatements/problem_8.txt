Given a signed 32-bit integer x, return x with its digits reversed. If reversing x causes the value to go outside the signed 32-bit integer range [-231, 231 - 1], then return 0. Assume the environment does not allow you to store 64-bit integers (signed or unsigned). <br><h4>Example 1:</h4> <br><h4>Input:</h4> x = 123 <br><h4>Output:</h4> 321 <br><h4>Example 2:</h4> <br><h4>Input:</h4> x = -123 <br><h4>Output:</h4> -321 <br><h4>Example 3:</h4> <br><h4>Input:</h4> x = 120 <br><h4>Output:</h4> 21 <br><h4>Constraints:</h4> -231 <= x <= 231 - 1
