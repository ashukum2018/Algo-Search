A binary tree is uni-valued if every node in the tree has the same value. Given the root of a binary tree, return true if the given tree is uni-valued, or false otherwise. <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [1,1,1,1,1,null,1] <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [2,2,2,5,2] <br><h4>Output:</h4> false <br><h4>Constraints:</h4> The number of nodes in the tree is in the range [1, 100]. 0 <= Node.val < 100
