A phrase is a palindrome if, after converting all uppercase letters into lowercase letters and removing all non-alphanumeric characters, it reads the same forward and backward. Alphanumeric characters include letters and numbers. Given a string s, return true if it is a palindrome, or false otherwise. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "A man, a plan, a canal:</h4> Panama" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> "amanaplanacanalpanama" is a palindrome. <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "race a car" <br><h4>Output:</h4> false <br><h4>Explanation:</h4> "raceacar" is not a palindrome. <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = " " <br><h4>Output:</h4> true <br><h4>Explanation:</h4> s is an empty string "" after removing non-alphanumeric characters. Since an empty string reads the same forward and backward, it is a palindrome. <br><h4>Constraints:</h4> 1 <= s.length <= 2 * 105 s consists only of printable ASCII characters.
