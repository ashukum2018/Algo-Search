(i.e., from left to right, then right to left for the next level and alternate between). <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [3,9,20,null,null,15,7] <br><h4>Output:</h4> [[3],[20,9],[15,7]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [1] <br><h4>Output:</h4> [[1]] <br><h4>Example 3:</h4> <br><h4>Input:</h4> root = [] <br><h4>Output:</h4> [] <br><h4>Constraints:</h4> The number of nodes in the tree is in the range [0, 2000]. -100 <= Node.val <= 100
