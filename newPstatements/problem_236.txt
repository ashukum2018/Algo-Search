Given the root of a binary tree, return all root-to-leaf paths in any order. A leaf is a node with no children. <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [1,2,3,null,5] <br><h4>Output:</h4> ["1->2->5","1->3"] <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [1] <br><h4>Output:</h4> ["1"] <br><h4>Constraints:</h4> The number of nodes in the tree is in the range [1, 100]. -100 <= Node.val <= 100
