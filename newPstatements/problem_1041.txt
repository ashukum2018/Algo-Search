Given a m x n matrix mat and an integer k, return a matrix answer where each answer[i][j] is the sum of all elements mat[r][c] for:</h4> i - k <= r <= i + k, j - k <= c <= j + k, and (r, c) is a valid position in the matrix. <br><h4>Example 1:</h4> <br><h4>Input:</h4> mat = [[1,2,3],[4,5,6],[7,8,9]], k = 1 <br><h4>Output:</h4> [[12,21,16],[27,45,33],[24,39,28]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> mat = [[1,2,3],[4,5,6],[7,8,9]], k = 2 <br><h4>Output:</h4> [[45,45,45],[45,45,45],[45,45,45]] <br><h4>Constraints:</h4> m == mat.length n == mat[i].length 1 <= m, n, k <= 100 1 <= mat[i][j] <= 100
