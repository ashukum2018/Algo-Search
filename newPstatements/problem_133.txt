Given a string s, partition s such that every substring of the partition is a palindrome. Return the minimum cuts needed for a palindrome partitioning of s. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "aab" <br><h4>Output:</h4> 1 <br><h4>Explanation:</h4> The palindrome partitioning ["aa","b"] could be produced using 1 cut. <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "a" <br><h4>Output:</h4> 0 <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "ab" <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 1 <= s.length <= 2000 s consists of lowercase English letters only.
