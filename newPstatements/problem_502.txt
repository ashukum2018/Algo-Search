You have k lists of sorted integers in non-decreasing order. Find the smallest range that includes at least one number from each of the k lists. We define the range [a, b] is smaller than range [c, d] if b - a < d - c or a < c if b - a == d - c. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [[4,10,15,24,26],[0,9,12,20],[5,18,22,30]] <br><h4>Output:</h4> [20,24] <br><h4>Explanation:</h4> List 1:</h4> [4, 10, 15, 24,26], 24 is in range [20,24]. List 2:</h4> [0, 9, 12, 20], 20 is in range [20,24]. List 3:</h4> [5, 18, 22, 30], 22 is in range [20,24]. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [[1,2,3],[1,2,3],[1,2,3]] <br><h4>Output:</h4> [1,1] <br><h4>Constraints:</h4> nums.length == k 1 <= k <= 3500 1 <= nums[i].length <= 50 -105 <= nums[i][j] <= 105 nums[i] is sorted in non-decreasing order.
