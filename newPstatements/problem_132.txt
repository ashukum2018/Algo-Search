Given a string s, partition s such that every substring of the partition is a palindrome. Return all possible palindrome partitioning of s. A palindrome string is a string that reads the same backward as forward. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "aab" <br><h4>Output:</h4> [["a","a","b"],["aa","b"]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "a" <br><h4>Output:</h4> [["a"]] <br><h4>Constraints:</h4> 1 <= s.length <= 16 s contains only lowercase English letters.
