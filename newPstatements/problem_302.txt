Given a set of distinct positive integers nums, return the largest subset answer such that every pair (answer[i], answer[j]) of elements in this subset satisfies:</h4> answer[i] % answer[j] == 0, or answer[j] % answer[i] == 0 If there are multiple solutions, return any of them. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,3] <br><h4>Output:</h4> [1,2] <br><h4>Explanation:</h4> [1,3] is also accepted. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,2,4,8] <br><h4>Output:</h4> [1,2,4,8] <br><h4>Constraints:</h4> 1 <= nums.length <= 1000 1 <= nums[i] <= 2 * 109 All the integers in nums are unique.
