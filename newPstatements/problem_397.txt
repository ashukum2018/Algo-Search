Given an integer n, return the largest palindromic integer that can be represented as the product of two n-digits integers. Since the answer can be very large, return it modulo 1337. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 2 <br><h4>Output:</h4> 987 <br><h4>Explanation:</h4> 99 x 91 = 9009, 9009 % 1337 = 987 <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 1 <br><h4>Output:</h4> 9 <br><h4>Constraints:</h4> 1 <= n <= 8
