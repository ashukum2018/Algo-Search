Given two arrays of integers with equal lengths, return the maximum value of:</h4> |arr1[i] - arr1[j]| + |arr2[i] - arr2[j]| + |i - j| where the maximum is taken over all 0 <= i, j < arr1.length. <br><h4>Example 1:</h4> <br><h4>Input:</h4> arr1 = [1,2,3,4], arr2 = [-1,4,5,6] <br><h4>Output:</h4> 13 <br><h4>Example 2:</h4> <br><h4>Input:</h4> arr1 = [1,-2,-5,0,10], arr2 = [0,-2,-1,-7,-4] <br><h4>Output:</h4> 20 <br><h4>Constraints:</h4> 2 <= arr1.length == arr2.length <= 40000 -10^6 <= arr1[i], arr2[i] <= 10^6
