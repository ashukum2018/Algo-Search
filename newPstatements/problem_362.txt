Given two integers n and k, return the kth lexicographically smallest integer in the range [1, n]. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 13, k = 2 <br><h4>Output:</h4> 10 <br><h4>Explanation:</h4> The lexicographical order is [1, 10, 11, 12, 13, 2, 3, 4, 5, 6, 7, 8, 9], so the second smallest number is 10. <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 1, k = 1 <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 1 <= k <= n <= 109
