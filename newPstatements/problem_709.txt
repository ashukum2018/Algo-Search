You are given two integer arrays nums1 and nums2 both of the same length. The advantage of nums1 with respect to nums2 is the number of indices i for which nums1[i] > nums2[i]. Return any permutation of nums1 that maximizes its advantage with respect to nums2. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums1 = [2,7,11,15], nums2 = [1,10,4,11] <br><h4>Output:</h4> [2,11,7,15] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums1 = [12,24,8,32], nums2 = [13,25,32,11] <br><h4>Output:</h4> [24,32,8,12] <br><h4>Constraints:</h4> 1 <= nums1.length <= 105 nums2.length == nums1.length 0 <= nums1[i], nums2[i] <= 109
