You are given an integer array nums. In one move, you can pick an index i where 0 <= i < nums.length and increment nums[i] by 1. Return the minimum number of moves to make every value in nums unique. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,2] <br><h4>Output:</h4> 1 <br><h4>Explanation:</h4> After 1 move, the array could be [1, 2, 3]. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [3,2,1,2,1,7] <br><h4>Output:</h4> 6 <br><h4>Explanation:</h4> After 6 moves, the array could be [3, 4, 1, 2, 5, 7]. It can be shown with 5 or less moves that it is impossible for the array to have all unique values. <br><h4>Constraints:</h4> 1 <= nums.length <= 105 0 <= nums[i] <= 105
