Given an input string (s) and a pattern (p), implement wildcard pattern matching with support for \'?\' and \'*\' where:</h4> \'?\' Matches any single character. \'*\' Matches any sequence of characters (including the empty sequence). The matching should cover the entire input string (not partial). <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "aa", p = "a" <br><h4>Output:</h4> false <br><h4>Explanation:</h4> "a" does not match the entire string "aa". <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "aa", p = "*" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> \'*\' matches any sequence. <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "cb", p = "?a" <br><h4>Output:</h4> false <br><h4>Explanation:</h4> \'?\' matches \'c\', but the second letter is \'a\', which does not match \'b\'. <br><h4>Constraints:</h4> 0 <= s.length, p.length <= 2000 s contains only lowercase English letters. p contains only lowercase English letters, \'?\' or \'*\'.
