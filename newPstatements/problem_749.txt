You are given an integer array nums and an integer k. For each index i where 0 <= i < nums.length, change nums[i] to be either nums[i] + k or nums[i] - k. The score of nums is the difference between the maximum and minimum elements in nums. Return the minimum score of nums after changing the values at each index. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1], k = 0 <br><h4>Output:</h4> 0 <br><h4>Explanation:</h4> The score is max(nums) - min(nums) = 1 - 1 = 0. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [0,10], k = 2 <br><h4>Output:</h4> 6 <br><h4>Explanation:</h4> Change nums to be [2, 8]. The score is max(nums) - min(nums) = 8 - 2 = 6. <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [1,3,6], k = 3 <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> Change nums to be [4, 6, 3]. The score is max(nums) - min(nums) = 6 - 3 = 3. <br><h4>Constraints:</h4> 1 <= nums.length <= 104 0 <= nums[i] <= 104 0 <= k <= 104
