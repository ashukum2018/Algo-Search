The Hamming distance between two integers is the number of positions at which the corresponding bits are different. Given two integers x and y, return the Hamming distance between them. <br><h4>Example 1:</h4> <br><h4>Input:</h4> x = 1, y = 4 <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> 1 (0 0 0 1) 4 (0 1 0 0) \xe2\x86\x91 \xe2\x86\x91 The above arrows point to positions where the corresponding bits are different. <br><h4>Example 2:</h4> <br><h4>Input:</h4> x = 3, y = 1 <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 0 <= x, y <= 231 - 1
