Given an integer array nums of size n, return the minimum number of moves required to make all array elements equal. In one move, you can increment n - 1 elements of the array by 1. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,3] <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> Only three moves are needed (remember each move increments two elements):</h4> [1,2,3] => [2,3,3] => [3,4,3] => [4,4,4] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,1,1] <br><h4>Output:</h4> 0 <br><h4>Constraints:</h4> n == nums.length 1 <= nums.length <= 105 -109 <= nums[i] <= 109 The answer is guaranteed to fit in a 32-bit integer.
