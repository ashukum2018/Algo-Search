Given a string s, check if it can be constructed by taking a substring of it and appending multiple copies of the substring together. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "abab" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> It is the substring "ab" twice. <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "aba" <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "abcabcabcabc" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> It is the substring "abc" four times or the substring "abcabc" twice. <br><h4>Constraints:</h4> 1 <= s.length <= 104 s consists of lowercase English letters.
