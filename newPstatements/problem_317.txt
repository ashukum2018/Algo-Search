Given an integer n, return all the numbers in the range [1, n] sorted in lexicographical order. You must write an algorithm that runs in O(n) time and uses O(1) extra space. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 13 <br><h4>Output:</h4> [1,10,11,12,13,2,3,4,5,6,7,8,9] <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 2 <br><h4>Output:</h4> [1,2] <br><h4>Constraints:</h4> 1 <= n <= 5 * 104
