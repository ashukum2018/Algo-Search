Given the root of a binary tree, return its maximum depth. A binary tree's maximum depth is the number of nodes along the longest path from the root node down to the farthest leaf node. <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [3,9,20,null,null,15,7] <br><h4>Output:</h4> 3 <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [1,null,2] <br><h4>Output:</h4> 2 <br><h4>Constraints:</h4> The number of nodes in the tree is in the range [0, 104]. -100 <= Node.val <= 100
