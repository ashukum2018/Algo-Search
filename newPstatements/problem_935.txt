Given an array nums of integers, a move consists of choosing any element and decreasing it by 1. An array A is a zigzag array if either:</h4> Every even-indexed element is greater than adjacent elements, ie. A[0] > A[1] < A[2] > A[3] < A[4] > ... OR, every odd-indexed element is greater than adjacent elements, ie. A[0] < A[1] > A[2] < A[3] > A[4] < ... Return the minimum number of moves to transform the given array nums into a zigzag array. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,3] <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> We can decrease 2 to 0 or 3 to 1. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [9,6,1,6,2] <br><h4>Output:</h4> 4 <br><h4>Constraints:</h4> 1 <= nums.length <= 1000 1 <= nums[i] <= 1000
