Implement pow(x, n), which calculates x raised to the power n (i.e., xn). <br><h4>Example 1:</h4> <br><h4>Input:</h4> x = 2.00000, n = 10 <br><h4>Output:</h4> 1024.00000 <br><h4>Example 2:</h4> <br><h4>Input:</h4> x = 2.10000, n = 3 <br><h4>Output:</h4> 9.26100 <br><h4>Example 3:</h4> <br><h4>Input:</h4> x = 2.00000, n = -2 <br><h4>Output:</h4> 0.25000 <br><h4>Explanation:</h4> 2-2 = 1/22 = 1/4 = 0.25 <br><h4>Constraints:</h4> -100.0 < x < 100.0 -231 <= n <= 231-1 -104 <= xn <= 104
