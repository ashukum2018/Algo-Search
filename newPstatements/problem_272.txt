You are given two integer arrays nums1 and nums2 of lengths m and n respectively. nums1 and nums2 represent the digits of two numbers. You are also given an integer k. Create the maximum number of length k <= m + n from digits of the two numbers. The relative order of the digits from the same array must be preserved. Return an array of the k digits representing the answer. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums1 = [3,4,6,5], nums2 = [9,1,2,5,8,3], k = 5 <br><h4>Output:</h4> [9,8,6,5,3] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums1 = [6,7], nums2 = [6,0,4], k = 5 <br><h4>Output:</h4> [6,7,6,0,4] <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums1 = [3,9], nums2 = [8,9], k = 3 <br><h4>Output:</h4> [9,8,9] <br><h4>Constraints:</h4> m == nums1.length n == nums2.length 1 <= m, n <= 500 0 <= nums1[i], nums2[i] <= 9 1 <= k <= m + n
