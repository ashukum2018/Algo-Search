An integer has sequential digits if and only if each digit in the number is one more than the previous digit. Return a sorted list of all the integers in the range [low, high] inclusive that have sequential digits. <br><h4>Example 1:</h4> <br><h4>Input:</h4> low = 100, high = 300 <br><h4>Output:</h4> [123,234] <br><h4>Example 2:</h4> <br><h4>Input:</h4> low = 1000, high = 13000 <br><h4>Output:</h4> [1234,2345,3456,4567,5678,6789,12345] <br><h4>Constraints:</h4> 10 <= low <= high <= 10^9
