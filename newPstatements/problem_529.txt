Nearly everyone has used the Multiplication Table. The multiplication table of size m x n is an integer matrix mat where mat[i][j] == i * j (1-indexed). Given three integers m, n, and k, return the kth smallest element in the m x n multiplication table. <br><h4>Example 1:</h4> <br><h4>Input:</h4> m = 3, n = 3, k = 5 <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> The 5th smallest number is 3. <br><h4>Example 2:</h4> <br><h4>Input:</h4> m = 2, n = 3, k = 6 <br><h4>Output:</h4> 6 <br><h4>Explanation:</h4> The 6th smallest number is 6. <br><h4>Constraints:</h4> 1 <= m, n <= 3 * 104 1 <= k <= m * n
