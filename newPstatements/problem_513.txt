Given a string s, return the number of palindromic substrings in it. A string is a palindrome when it reads the same backward as forward. A substring is a contiguous sequence of characters within the string. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "abc" <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> Three palindromic strings:</h4> "a", "b", "c". <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "aaa" <br><h4>Output:</h4> 6 <br><h4>Explanation:</h4> Six palindromic strings:</h4> "a", "a", "a", "aa", "aa", "aaa". <br><h4>Constraints:</h4> 1 <= s.length <= 1000 s consists of lowercase English letters.
