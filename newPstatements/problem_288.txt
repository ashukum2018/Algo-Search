Given an integer n, return true if it is a power of four. Otherwise, return false. An integer n is a power of four, if there exists an integer x such that n == 4x. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 16 <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 5 <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = 1 <br><h4>Output:</h4> true <br><h4>Constraints:</h4> -231 <= n <= 231 - 1 Follow up:</h4> Could you solve it without loops/recursion?
