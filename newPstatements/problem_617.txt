In a string composed of \'L\', \'R\', and \'X\' characters, like "RXXLRXRXL", a move consists of either replacing one occurrence of "XL" with "LX", or replacing one occurrence of "RX" with "XR". Given the starting string start and the ending string end, return True if and only if there exists a sequence of moves to transform one string to the other. <br><h4>Example 1:</h4> <br><h4>Input:</h4> start = "RXXLRXRXL", end = "XRLXXRRLX" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> We can transform start to end following these steps:</h4> RXXLRXRXL -> XRXLRXRXL -> XRLXRXRXL -> XRLXXRRXL -> XRLXXRRLX <br><h4>Example 2:</h4> <br><h4>Input:</h4> start = "X", end = "L" <br><h4>Output:</h4> false <br><h4>Constraints:</h4> 1 <= start.length <= 104 start.length == end.length Both start and end will only consist of characters in \'L\', \'R\', and \'X\'.
