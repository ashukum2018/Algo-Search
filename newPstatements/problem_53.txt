The n-queens puzzle is the problem of placing n queens on an n x n chessboard such that no two queens attack each other. Given an integer n, return the number of distinct solutions to the n-queens puzzle. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 4 <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> There are two distinct solutions to the 4-queens puzzle as shown. <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 1 <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 1 <= n <= 9
