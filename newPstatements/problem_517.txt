Given the root of a binary tree, return all duplicate subtrees. For each kind of duplicate subtrees, you only need to return the root node of any one of them. Two trees are duplicate if they have the same structure with the same node values. <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [1,2,3,4,null,2,4,null,null,4] <br><h4>Output:</h4> [[2,4],[4]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [2,1,1] <br><h4>Output:</h4> [[1]] <br><h4>Example 3:</h4> <br><h4>Input:</h4> root = [2,2,2,3,null,3,null] <br><h4>Output:</h4> [[2,3],[3]] <br><h4>Constraints:</h4> The number of the nodes in the tree will be in the range [1, 10^4] -200 <= Node.val <= 200
