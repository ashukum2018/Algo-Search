Given a list of dominoes, dominoes[i] = [a, b] is equivalent to dominoes[j] = [c, d] if and only if either (a == c and b == d), or (a == d and b == c) - that is, one domino can be rotated to be equal to another domino. Return the number of pairs (i, j) for which 0 <= i < j < dominoes.length, and dominoes[i] is equivalent to dominoes[j]. <br><h4>Example 1:</h4> <br><h4>Input:</h4> dominoes = [[1,2],[2,1],[3,4],[5,6]] <br><h4>Output:</h4> 1 <br><h4>Example 2:</h4> <br><h4>Input:</h4> dominoes = [[1,2],[1,2],[1,1],[1,2],[2,2]] <br><h4>Output:</h4> 3 <br><h4>Constraints:</h4> 1 <= dominoes.length <= 4 * 104 dominoes[i].length == 2 1 <= dominoes[i][j] <= 9
