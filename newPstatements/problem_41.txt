Given a collection of candidate numbers (candidates) and a target number (target), find all unique combinations in candidates where the candidate numbers sum to target. Each number in candidates may only be used once in the combination. Note:</h4> The solution set must not contain duplicate combinations. <br><h4>Example 1:</h4> <br><h4>Input:</h4> candidates = [10,1,2,7,6,1,5], target = 8 <br><h4>Output:</h4> [ [1,1,6], [1,2,5], [1,7], [2,6] ] <br><h4>Example 2:</h4> <br><h4>Input:</h4> candidates = [2,5,2,1,2], target = 5 <br><h4>Output:</h4> [ [1,2,2], [5] ] <br><h4>Constraints:</h4> 1 <= candidates.length <= 100 1 <= candidates[i] <= 50 1 <= target <= 30
