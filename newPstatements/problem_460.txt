Given an integer array nums of 2n integers, group these integers into n pairs (a1, b1), (a2, b2), ..., (an, bn) such that the sum of min(ai, bi) for all i is maximized. Return the maximized sum. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,4,3,2] <br><h4>Output:</h4> 4 <br><h4>Explanation:</h4> All possible pairings (ignoring the ordering of elements) are:</h4> 1. (1, 4), (2, 3) -> min(1, 4) + min(2, 3) = 1 + 2 = 3 2. (1, 3), (2, 4) -> min(1, 3) + min(2, 4) = 1 + 2 = 3 3. (1, 2), (3, 4) -> min(1, 2) + min(3, 4) = 1 + 3 = 4 So the maximum possible sum is 4. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [6,2,6,5,1,2] <br><h4>Output:</h4> 9 <br><h4>Explanation:</h4> The optimal pairing is (2, 1), (2, 5), (6, 6). min(2, 1) + min(2, 5) + min(6, 6) = 1 + 2 + 6 = 9. <br><h4>Constraints:</h4> 1 <= n <= 104 nums.length == 2 * n -104 <= nums[i] <= 104
