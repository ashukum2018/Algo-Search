Given an integer array nums, return the maximum result of nums[i] XOR nums[j], where 0 <= i <= j < n. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [3,10,5,25,2,8] <br><h4>Output:</h4> 28 <br><h4>Explanation:</h4> The maximum result is 5 XOR 25 = 28. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [14,70,53,83,49,91,36,80,92,51,66,70] <br><h4>Output:</h4> 127 <br><h4>Constraints:</h4> 1 <= nums.length <= 2 * 105 0 <= nums[i] <= 231 - 1
