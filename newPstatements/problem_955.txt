With respect to a given puzzle string, a word is valid if both the following conditions are satisfied:</h4> word contains the first letter of puzzle. For each letter in word, that letter is in puzzle. \t For example, if the puzzle is "abcdefg", then valid words are "faced", "cabbage", and "baggage", while invalid words are "beefed" (does not include \'a\') and "based" (includes \'s\' which is not in the puzzle). Each puzzles[i] does not contain repeated characters.
