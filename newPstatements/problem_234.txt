Given a string expression of numbers and operators, return all possible results from computing all the different possible ways to group numbers and operators. You may return the answer in any order. <br><h4>Example 1:</h4> <br><h4>Input:</h4> expression = "2-1-1" <br><h4>Output:</h4> [0,2] <br><h4>Explanation:</h4> ((2-1)-1) = 0 (2-(1-1)) = 2 <br><h4>Example 2:</h4> <br><h4>Input:</h4> expression = "2*3-4*5" <br><h4>Output:</h4> [-34,-14,-10,-10,10] <br><h4>Explanation:</h4> (2*(3-(4*5))) = -34 ((2*3)-(4*5)) = -14 ((2*(3-4))*5) = -10 (2*((3-4)*5)) = -10 (((2*3)-4)*5) = 10 <br><h4>Constraints:</h4> 1 <= expression.length <= 20 expression consists of digits and the operator \'+\', \'-\', and \'*\'. All the integer values in the input expression are in the range [0, 99].
