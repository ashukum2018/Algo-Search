Given the head of a singly linked list, return true if it is a palindrome. <br><h4>Example 1:</h4> <br><h4>Input:</h4> head = [1,2,2,1] <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> head = [1,2] <br><h4>Output:</h4> false <br><h4>Constraints:</h4> The number of nodes in the list is in the range [1, 105]. 0 <= Node.val <= 9 Follow up:</h4> Could you do it in O(n) time and O(1) space?
