Given a sorted array of distinct integers and a target value, return the index if the target is found. If not, return the index where it would be if it were inserted in order. You must write an algorithm with O(log n) runtime complexity. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,3,5,6], target = 5 <br><h4>Output:</h4> 2 <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,3,5,6], target = 2 <br><h4>Output:</h4> 1 <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [1,3,5,6], target = 7 <br><h4>Output:</h4> 4 <br><h4>Constraints:</h4> 1 <= nums.length <= 104 -104 <= nums[i] <= 104 nums contains distinct values sorted in ascending order. -104 <= target <= 104
