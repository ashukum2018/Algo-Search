Given an array of digits which is sorted in non-decreasing order. You can write numbers using each digits[i] as many times as we want. For example, if digits = [\'1\',\'3\',\'5\'], we may write numbers such as \'13\', \'551\', and \'1351315\'. Return the number of positive integers that can be generated that are less than or equal to a given integer n. <br><h4>Example 1:</h4> <br><h4>Input:</h4> digits = ["1","3","5","7"], n = 100 <br><h4>Output:</h4> 20 <br><h4>Explanation:</h4> The 20 numbers that can be written are:</h4> 1, 3, 5, 7, 11, 13, 15, 17, 31, 33, 35, 37, 51, 53, 55, 57, 71, 73, 75, 77. <br><h4>Example 2:</h4> <br><h4>Input:</h4> digits = ["1","4","9"], n = 1000000000 <br><h4>Output:</h4> 29523 <br><h4>Explanation:</h4> We can write 3 one digit numbers, 9 two digit numbers, 27 three digit numbers, 81 four digit numbers, 243 five digit numbers, 729 six digit numbers, 2187 seven digit numbers, 6561 eight digit numbers, and 19683 nine digit numbers. In total, this is 29523 integers that can be written using the digits array. <br><h4>Example 3:</h4> <br><h4>Input:</h4> digits = ["7"], n = 8 <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 1 <= digits.length <= 9 digits[i].length == 1 digits[i] is a digit from \'1\' to \'9\'. All the values in digits are unique. digits is sorted in non-decreasing order. 1 <= n <= 109
