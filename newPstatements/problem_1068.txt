Given an array arr of integers, check if there exists two integers N and M such that N is the double of M ( i.e. N = 2 * M). More formally check if there exists two indices i and j such that :</h4> i != j 0 <= i, j < arr.length arr[i] == 2 * arr[j] <br><h4>Example 1:</h4> <br><h4>Input:</h4> arr = [10,2,5,3] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> N = 10 is the double of M = 5,that is, 10 = 2 * 5. <br><h4>Example 2:</h4> <br><h4>Input:</h4> arr = [7,1,14,11] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> N = 14 is the double of M = 7,that is, 14 = 2 * 7. <br><h4>Example 3:</h4> <br><h4>Input:</h4> arr = [3,1,7,11] <br><h4>Output:</h4> false <br><h4>Explanation:</h4> In this case does not exist N and M, such that N = 2 * M. <br><h4>Constraints:</h4> 2 <= arr.length <= 500 -10^3 <= arr[i] <= 10^3
