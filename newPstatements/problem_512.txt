You are given an array of n pairs pairs where pairs[i] = [lefti, righti] and lefti < righti. A pair p2 = [c, d] follows a pair p1 = [a, b] if b < c. A chain of pairs can be formed in this fashion. Return the length longest chain which can be formed. You do not need to use up all the given intervals. You can select pairs in any order. <br><h4>Example 1:</h4> <br><h4>Input:</h4> pairs = [[1,2],[2,3],[3,4]] <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> The longest chain is [1,2] -> [3,4]. <br><h4>Example 2:</h4> <br><h4>Input:</h4> pairs = [[1,2],[7,8],[4,5]] <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> The longest chain is [1,2] -> [4,5] -> [7,8]. <br><h4>Constraints:</h4> n == pairs.length 1 <= n <= 1000 -1000 <= lefti < righti <= 1000
