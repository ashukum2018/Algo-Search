Given an integer array nums, reorder it such that nums[0] < nums[1] > nums[2] < nums[3].... You may assume the input array always has a valid answer. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,5,1,1,6,4] <br><h4>Output:</h4> [1,6,1,5,1,4] <br><h4>Explanation:</h4> [1,4,1,5,1,6] is also accepted. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,3,2,2,3,1] <br><h4>Output:</h4> [2,3,1,3,1,2] <br><h4>Constraints:</h4> 1 <= nums.length <= 5 * 104 0 <= nums[i] <= 5000 It is guaranteed that there will be an answer for the given input nums. Follow Up:</h4> Can you do it in O(n) time and/or in-place with O(1) extra space?
