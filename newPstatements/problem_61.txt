The set [1, 2, 3, ..., n] contains a total of n! unique permutations. By listing and labeling all of the permutations in order, we get the following sequence for n = 3:</h4> "123" "132" "213" "231" "312" "321" Given n and k, return the kth permutation sequence. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 3, k = 3 <br><h4>Output:</h4> "213" <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 4, k = 9 <br><h4>Output:</h4> "2314" <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = 3, k = 1 <br><h4>Output:</h4> "123" <br><h4>Constraints:</h4> 1 <= n <= 9 1 <= k <= n!
