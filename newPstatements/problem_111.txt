Given a binary tree, determine if it is height-balanced. For this problem, a height-balanced binary tree is defined as:</h4> a binary tree in which the left and right subtrees of every node differ in height by no more than 1. <br><h4>Example 1:</h4> <br><h4>Input:</h4> root = [3,9,20,null,null,15,7] <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> root = [1,2,2,3,3,null,null,4,4] <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> root = [] <br><h4>Output:</h4> true <br><h4>Constraints:</h4> The number of nodes in the tree is in the range [0, 5000]. -104 <= Node.val <= 104
