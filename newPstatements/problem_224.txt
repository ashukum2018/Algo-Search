Given an integer n, return true if it is a power of two. Otherwise, return false. An integer n is a power of two, if there exists an integer x such that n == 2x. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 1 <br><h4>Output:</h4> true <br><h4>Explanation:</h4> 20 = 1 <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 16 <br><h4>Output:</h4> true <br><h4>Explanation:</h4> 24 = 16 <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = 3 <br><h4>Output:</h4> false <br><h4>Constraints:</h4> -231 <= n <= 231 - 1 Follow up:</h4> Could you solve it without loops/recursion?
