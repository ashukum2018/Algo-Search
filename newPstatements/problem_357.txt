Given a string s, return the number of segments in the string. A segment is defined to be a contiguous sequence of non-space characters. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "Hello, my name is John" <br><h4>Output:</h4> 5 <br><h4>Explanation:</h4> The five segments are ["Hello,", "my", "name", "is", "John"] <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "Hello" <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 0 <= s.length <= 300 s consists of lowercase and uppercase English letters, digits, or one of the following characters "!@#$%^&*()_+-=\',.:</h4>". The only space character in s is \' \'.
