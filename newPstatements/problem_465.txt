Given two strings s1 and s2, return true if s2 contains a permutation of s1, or false otherwise. In other words, return true if one of s1\'s permutations is the substring of s2. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s1 = "ab", s2 = "eidbaooo" <br><h4>Output:</h4> true <br><h4>Explanation:</h4> s2 contains one permutation of s1 ("ba"). <br><h4>Example 2:</h4> <br><h4>Input:</h4> s1 = "ab", s2 = "eidboaoo" <br><h4>Output:</h4> false <br><h4>Constraints:</h4> 1 <= s1.length, s2.length <= 104 s1 and s2 consist of lowercase English letters.
