Given a pattern and a string s, find if s follows the same pattern. Here follow means a full match, such that there is a bijection between a letter in pattern and a non-empty word in s. <br><h4>Example 1:</h4> <br><h4>Input:</h4> pattern = "abba", s = "dog cat cat dog" <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> pattern = "abba", s = "dog cat cat fish" <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> pattern = "aaaa", s = "dog cat cat dog" <br><h4>Output:</h4> false <br><h4>Constraints:</h4> 1 <= pattern.length <= 300 pattern contains only lower-case English letters. 1 <= s.length <= 3000 s contains only lowercase English letters and spaces \' \'. s does not contain any leading or trailing spaces. All the words in s are separated by a single space.
