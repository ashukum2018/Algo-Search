Given a m x n matrix grid which is sorted in non-increasing order both row-wise and column-wise, return the number of negative numbers in grid. <br><h4>Example 1:</h4> <br><h4>Input:</h4> grid = [[4,3,2,-1],[3,2,1,-1],[1,1,-1,-2],[-1,-1,-2,-3]] <br><h4>Output:</h4> 8 <br><h4>Explanation:</h4> There are 8 negatives number in the matrix. <br><h4>Example 2:</h4> <br><h4>Input:</h4> grid = [[3,2],[1,0]] <br><h4>Output:</h4> 0 <br><h4>Constraints:</h4> m == grid.length n == grid[i].length 1 <= m, n <= 100 -100 <= grid[i][j] <= 100 Follow up:</h4> Could you find an O(n + m) solution?
