Given an integer array nums and an integer k, return the number of good subarrays of nums. A good array is an array where the number of different integers in that array is exactly k. For example, [1,2,3,1,2] has 3 different integers:</h4> 1, 2, and 3. A subarray is a contiguous part of an array. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,1,2,3], k = 2 <br><h4>Output:</h4> 7 <br><h4>Explanation:</h4> Subarrays formed with exactly 2 different integers:</h4> [1,2], [2,1], [1,2], [2,3], [1,2,1], [2,1,2], [1,2,1,2] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,2,1,3,4], k = 3 <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> Subarrays formed with exactly 3 different integers:</h4> [1,2,1,3], [2,1,3], [1,3,4]. <br><h4>Constraints:</h4> 1 <= nums.length <= 2 * 104 1 <= nums[i], k <= nums.length
