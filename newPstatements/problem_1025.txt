Given an array of integers nums and a positive integer k, check whether it is possible to divide this array into sets of k consecutive numbers. Return true if it is possible. Otherwise, return false. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,3,3,4,4,5,6], k = 4 <br><h4>Output:</h4> true <br><h4>Explanation:</h4> Array can be divided into [1,2,3,4] and [3,4,5,6]. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [3,2,1,2,3,4,3,4,5,9,10,11], k = 3 <br><h4>Output:</h4> true <br><h4>Explanation:</h4> Array can be divided into [1,2,3] , [2,3,4] , [3,4,5] and [9,10,11]. <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [1,2,3,4], k = 3 <br><h4>Output:</h4> false <br><h4>Explanation:</h4> Each array should be divided in subarrays of size 3. <br><h4>Constraints:</h4> 1 <= k <= nums.length <= 105 1 <= nums[i] <= 109 Note:</h4> This question is the same as 846:</h4> https:</h4>//leetcode.com/problems/hand-of-straights/
