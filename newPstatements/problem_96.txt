Given an integer n, return all the structurally unique BST's (binary search trees), which has exactly n nodes of unique values from 1 to n. Return the answer in any order. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 3 <br><h4>Output:</h4> [[1,null,2,null,3],[1,null,3,2],[2,1,3],[3,1,null,null,2],[3,2,null,1]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 1 <br><h4>Output:</h4> [[1]] <br><h4>Constraints:</h4> 1 <= n <= 8
