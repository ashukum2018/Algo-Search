Given an integer array of even length arr, return true if it is possible to reorder arr such that arr[2 * i + 1] = 2 * arr[2 * i] for every 0 <= i < len(arr) / 2, or false otherwise. <br><h4>Example 1:</h4> <br><h4>Input:</h4> arr = [3,1,3,6] <br><h4>Output:</h4> false <br><h4>Example 2:</h4> <br><h4>Input:</h4> arr = [2,1,2,6] <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> arr = [4,-2,2,-4] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> We can take two groups, [-2,-4] and [2,4] to form [-2,-4,2,4] or [2,4,-2,-4]. <br><h4>Constraints:</h4> 2 <= arr.length <= 3 * 104 arr.length is even. -105 <= arr[i] <= 105
