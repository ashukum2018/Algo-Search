Given an array of integers arr, return true if we can partition the array into three non-empty parts with equal sums. Formally, we can partition the array if we can find indexes i + 1 < j with (arr[0] + arr[1] + ... + arr[i] == arr[i + 1] + arr[i + 2] + ... + arr[j - 1] == arr[j] + arr[j + 1] + ... + arr[arr.length - 1]) <br><h4>Example 1:</h4> <br><h4>Input:</h4> arr = [0,2,1,-6,6,-7,9,1,2,0,1] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> 0 + 2 + 1 = -6 + 6 - 7 + 9 + 1 = 2 + 0 + 1 <br><h4>Example 2:</h4> <br><h4>Input:</h4> arr = [0,2,1,-6,6,7,9,-1,2,0,1] <br><h4>Output:</h4> false <br><h4>Example 3:</h4> <br><h4>Input:</h4> arr = [3,3,6,5,-2,2,5,1,-9,4] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> 3 + 3 = 6 = 5 - 2 + 2 + 5 + 1 - 9 + 4 <br><h4>Constraints:</h4> 3 <= arr.length <= 5 * 104 -104 <= arr[i] <= 104
