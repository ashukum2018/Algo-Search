Given a string s. In one step you can insert any character at any index of the string. Return the minimum number of steps to make s palindrome. A Palindrome String is one that reads the same backward as well as forward. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "zzazz" <br><h4>Output:</h4> 0 <br><h4>Explanation:</h4> The string "zzazz" is already palindrome we don\'t need any insertions. <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "mbadm" <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> String can be "mbdadbm" or "mdbabdm". <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "leetcode" <br><h4>Output:</h4> 5 <br><h4>Explanation:</h4> Inserting 5 characters the string becomes "leetcodocteel". <br><h4>Constraints:</h4> 1 <= s.length <= 500 s consists of lowercase English letters.
