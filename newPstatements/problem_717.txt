A positive integer is magical if it is divisible by either a or b. Given the three integers n, a, and b, return the nth magical number. Since the answer may be very large, return it modulo 109 + 7. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 1, a = 2, b = 3 <br><h4>Output:</h4> 2 <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 4, a = 2, b = 3 <br><h4>Output:</h4> 6 <br><h4>Constraints:</h4> 1 <= n <= 109 2 <= a, b <= 4 * 104
