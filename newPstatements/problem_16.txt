Given an integer array nums, return all the triplets [nums[i], nums[j], nums[k]] such that i != j, i != k, and j != k, and nums[i] + nums[j] + nums[k] == 0. Notice that the solution set must not contain duplicate triplets. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [-1,0,1,2,-1,-4] <br><h4>Output:</h4> [[-1,-1,2],[-1,0,1]] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [] <br><h4>Output:</h4> [] <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [0] <br><h4>Output:</h4> [] <br><h4>Constraints:</h4> 0 <= nums.length <= 3000 -105 <= nums[i] <= 105
