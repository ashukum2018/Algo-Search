Given a valid (IPv4) IP address, return a defanged version of that IP address. A defanged IP address replaces every period "." with "[.]". <br><h4>Example 1:</h4> <br><h4>Input:</h4> address = "1.1.1.1" <br><h4>Output:</h4> "1[.]1[.]1[.]1" <br><h4>Example 2:</h4> <br><h4>Input:</h4> address = "255.100.50.0" <br><h4>Output:</h4> "255[.]100[.]50[.]0" <br><h4>Constraints:</h4> The given address is a valid IPv4 address.
