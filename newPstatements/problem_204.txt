Design a data structure that supports adding new words and finding if a string matches any previously added string. Implement the WordDictionary class:</h4> WordDictionary() Initializes the object. void addWord(word) Adds word to the data structure, it can be matched later. bool search(word) Returns true if there is any string in the data structure that matches word or false otherwise. word may contain dots \'.\' where dots can be matched with any letter. <br><h4>Example:</h4> <br><h4>Input ["WordDictionary","addWord","addWord","addWord","search","search","search","search"] [[],["bad"],["dad"],["mad"],["pad"],["bad"],[".ad"],["b.."]] <br><h4>Output [null,null,null,null,false,true,true,true] <br><h4>Explanation WordDictionary wordDictionary = new WordDictionary(); wordDictionary.addWord("bad"); wordDictionary.addWord("dad"); wordDictionary.addWord("mad"); wordDictionary.search("pad"); // return False wordDictionary.search("bad"); // return True wordDictionary.search(".ad"); // return True wordDictionary.search("b.."); // return True <br><h4>Constraints:</h4> 1 <= word.length <= 25 word in addWord consists of lowercase English letters. word in search consist of \'.\' or lowercase English letters. There will be at most 3 dots in word for search queries. At most 104 calls will be made to addWord and search.
