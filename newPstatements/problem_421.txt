The Fibonacci numbers, commonly denoted F(n) form a sequence, called the Fibonacci sequence, such that each number is the sum of the two preceding ones, starting from 0 and 1. That is, F(0) = 0, F(1) = 1 F(n) = F(n - 1) + F(n - 2), for n > 1. Given n, calculate F(n). <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 2 <br><h4>Output:</h4> 1 <br><h4>Explanation:</h4> F(2) = F(1) + F(0) = 1 + 0 = 1. <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 3 <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> F(3) = F(2) + F(1) = 1 + 1 = 2. <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = 4 <br><h4>Output:</h4> 3 <br><h4>Explanation:</h4> F(4) = F(3) + F(2) = 2 + 1 = 3. <br><h4>Constraints:</h4> 0 <= n <= 30
