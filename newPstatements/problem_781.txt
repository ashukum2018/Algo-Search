A permutation perm of n + 1 integers of all the integers in the range [0, n] can be represented as a string s of length n where:</h4> s[i] == \'I\' if perm[i] < perm[i + 1], and s[i] == \'D\' if perm[i] > perm[i + 1]. Given a string s, reconstruct the permutation perm and return it. If there are multiple valid permutations perm, return any of them. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "IDID" <br><h4>Output:</h4> [0,4,1,3,2] <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "III" <br><h4>Output:</h4> [0,1,2,3] <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "DDI" <br><h4>Output:</h4> [3,2,0,1] <br><h4>Constraints:</h4> 1 <= s.length <= 105 s[i] is either \'I\' or \'D\'.
