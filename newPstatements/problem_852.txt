You are given an integer array values where values[i] represents the value of the ith sightseeing spot. Two sightseeing spots i and j have a distance j - i between them. The score of a pair (i < j) of sightseeing spots is values[i] + values[j] + i - j:</h4> the sum of the values of the sightseeing spots, minus the distance between them. Return the maximum score of a pair of sightseeing spots. <br><h4>Example 1:</h4> <br><h4>Input:</h4> values = [8,1,5,2,6] <br><h4>Output:</h4> 11 <br><h4>Explanation:</h4> i = 0, j = 2, values[i] + values[j] + i - j = 8 + 5 + 0 - 2 = 11 <br><h4>Example 2:</h4> <br><h4>Input:</h4> values = [1,2] <br><h4>Output:</h4> 2 <br><h4>Constraints:</h4> 2 <= values.length <= 5 * 104 1 <= values[i] <= 1000
