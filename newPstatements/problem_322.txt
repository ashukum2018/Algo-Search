Given an array rectangles where rectangles[i] = [xi, yi, ai, bi] represents an axis-aligned rectangle. The bottom-left point of the rectangle is (xi, yi) and the top-right point of it is (ai, bi). Return true if all the rectangles together form an exact cover of a rectangular region. <br><h4>Example 1:</h4> <br><h4>Input:</h4> rectangles = [[1,1,3,3],[3,1,4,2],[3,2,4,4],[1,3,2,4],[2,3,3,4]] <br><h4>Output:</h4> true <br><h4>Explanation:</h4> All 5 rectangles together form an exact cover of a rectangular region. <br><h4>Example 2:</h4> <br><h4>Input:</h4> rectangles = [[1,1,2,3],[1,3,2,4],[3,1,4,2],[3,2,4,4]] <br><h4>Output:</h4> false <br><h4>Explanation:</h4> Because there is a gap between the two rectangular regions. <br><h4>Example 3:</h4> <br><h4>Input:</h4> rectangles = [[1,1,3,3],[3,1,4,2],[1,3,2,4],[2,2,4,4]] <br><h4>Output:</h4> false <br><h4>Explanation:</h4> Because two of the rectangles overlap with each other. <br><h4>Constraints:</h4> 1 <= rectangles.length <= 2 * 104 rectangles[i].length == 4 -105 <= xi, yi, ai, bi <= 105
