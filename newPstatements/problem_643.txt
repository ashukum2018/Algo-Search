International Morse Code defines a standard encoding where each letter is mapped to a series of dots and dashes, as follows:</h4> \'a\' maps to ".-", \'b\' maps to "-...", \'c\' maps to "-.-.", and so on. For convenience, the full table for the 26 letters of the English alphabet is given below:</h4> [".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."] Given an array of strings words where each word can be written as a concatenation of the Morse code of each letter. For example, "cab" can be written as "-.-..--...", which is the concatenation of "-.-.", ".-", and "-...". We will call such a concatenation the transformation of a word. Return the number of different transformations among all words we have. <br><h4>Example 1:</h4> <br><h4>Input:</h4> words = ["gin","zen","gig","msg"] <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> The transformation of each word is:</h4> "gin" -> "--...-." "zen" -> "--...-." "gig" -> "--...--." "msg" -> "--...--." There are 2 different transformations:</h4> "--...-." and "--...--.". <br><h4>Example 2:</h4> <br><h4>Input:</h4> words = ["a"] <br><h4>Output:</h4> 1 <br><h4>Constraints:</h4> 1 <= words.length <= 100 1 <= words[i].length <= 12 words[i] consists of lowercase English letters.
