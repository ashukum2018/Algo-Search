Given an integer n represented as a string, return the smallest good base of n. We call k >= 2 a good base of n, if all digits of n base k are 1\'s. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = "13" <br><h4>Output:</h4> "3" <br><h4>Explanation:</h4> 13 base 3 is 111. <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = "4681" <br><h4>Output:</h4> "8" <br><h4>Explanation:</h4> 4681 base 8 is 11111. <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = "1000000000000000000" <br><h4>Output:</h4> "999999999999999999" <br><h4>Explanation:</h4> 1000000000000000000 base 999999999999999999 is 11. <br><h4>Constraints:</h4> n is an integer in the range [3, 1018]. n does not contain any leading zeros.
