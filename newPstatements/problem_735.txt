An array is monotonic if it is either monotone increasing or monotone decreasing. An array nums is monotone increasing if for all i <= j, nums[i] <= nums[j]. An array nums is monotone decreasing if for all i <= j, nums[i] >= nums[j]. Given an integer array nums, return true if the given array is monotonic, or false otherwise. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,2,2,3] <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [6,5,4,4] <br><h4>Output:</h4> true <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [1,3,2] <br><h4>Output:</h4> false <br><h4>Constraints:</h4> 1 <= nums.length <= 105 -105 <= nums[i] <= 105
