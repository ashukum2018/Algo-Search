For two strings s and t, we say "t divides s" if and only if s = t + ... + t (i.e., t is concatenated with itself one or more times). Given two strings str1 and str2, return the largest string x such that x divides both str1 and str2. <br><h4>Example 1:</h4> <br><h4>Input:</h4> str1 = "ABCABC", str2 = "ABC" <br><h4>Output:</h4> "ABC" <br><h4>Example 2:</h4> <br><h4>Input:</h4> str1 = "ABABAB", str2 = "ABAB" <br><h4>Output:</h4> "AB" <br><h4>Example 3:</h4> <br><h4>Input:</h4> str1 = "LEET", str2 = "CODE" <br><h4>Output:</h4> "" <br><h4>Constraints:</h4> 1 <= str1.length, str2.length <= 1000 str1 and str2 consist of English uppercase letters.
