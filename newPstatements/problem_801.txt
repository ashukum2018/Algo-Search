A ramp in an integer array nums is a pair (i, j) for which i < j and nums[i] <= nums[j]. The width of such a ramp is j - i. Given an integer array nums, return the maximum width of a ramp in nums. If there is no ramp in nums, return 0. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [6,0,8,2,1,5] <br><h4>Output:</h4> 4 <br><h4>Explanation:</h4> The maximum width ramp is achieved at (i, j) = (1, 5):</h4> nums[1] = 0 and nums[5] = 5. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [9,8,1,0,1,9,4,0,4,1] <br><h4>Output:</h4> 7 <br><h4>Explanation:</h4> The maximum width ramp is achieved at (i, j) = (2, 9):</h4> nums[2] = 1 and nums[9] = 1. <br><h4>Constraints:</h4> 2 <= nums.length <= 5 * 104 0 <= nums[i] <= 5 * 104
