Given an array of integers arr. We want to select three indices i, j and k where (0 <= i < j <= k < arr.length). Let's define a and b as follows:</h4> a = arr[i] ^ arr[i + 1] ^ ... ^ arr[j - 1] b = arr[j] ^ arr[j + 1] ^ ... ^ arr[k] Note that ^ denotes the bitwise-xor operation. Return the number of triplets (i, j and k) Where a == b. <br><h4>Example 1:</h4> <br><h4>Input:</h4> arr = [2,3,1,6,7] <br><h4>Output:</h4> 4 <br><h4>Explanation:</h4> The triplets are (0,1,2), (0,2,2), (2,3,4) and (2,4,4) <br><h4>Example 2:</h4> <br><h4>Input:</h4> arr = [1,1,1,1,1] <br><h4>Output:</h4> 10 <br><h4>Constraints:</h4> 1 <= arr.length <= 300 1 <= arr[i] <= 108
