Given an integer array nums, you need to find one continuous subarray that if you only sort this subarray in ascending order, then the whole array will be sorted in ascending order. Return the shortest such subarray and output its length. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [2,6,4,8,10,9,15] <br><h4>Output:</h4> 5 <br><h4>Explanation:</h4> You need to sort [6, 4, 8, 10, 9] in ascending order to make the whole array sorted in ascending order. <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [1,2,3,4] <br><h4>Output:</h4> 0 <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [1] <br><h4>Output:</h4> 0 <br><h4>Constraints:</h4> 1 <= nums.length <= 104 -105 <= nums[i] <= 105 Follow up:</h4> Can you solve it in O(n) time complexity?
