A string is called a happy prefix if is a non-empty prefix which is also a suffix (excluding itself). Given a string s, return the longest happy prefix of s. Return an empty string "" if no such prefix exists. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "level" <br><h4>Output:</h4> "l" <br><h4>Explanation:</h4> s contains 4 prefix excluding itself ("l", "le", "lev", "leve"), and suffix ("l", "el", "vel", "evel"). The largest prefix which is also suffix is given by "l". <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "ababab" <br><h4>Output:</h4> "abab" <br><h4>Explanation:</h4> "abab" is the largest prefix which is also suffix. They can overlap in the original string. <br><h4>Constraints:</h4> 1 <= s.length <= 105 s contains only lowercase English letters.
