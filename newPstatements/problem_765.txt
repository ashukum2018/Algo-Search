A binary string is monotone increasing if it consists of some number of 0\'s (possibly none), followed by some number of 1\'s (also possibly none). You are given a binary string s. You can flip s[i] changing it from 0 to 1 or from 1 to 0. Return the minimum number of flips to make s monotone increasing. <br><h4>Example 1:</h4> <br><h4>Input:</h4> s = "00110" <br><h4>Output:</h4> 1 <br><h4>Explanation:</h4> We flip the last digit to get 00111. <br><h4>Example 2:</h4> <br><h4>Input:</h4> s = "010110" <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> We flip to get 011111, or alternatively 000111. <br><h4>Example 3:</h4> <br><h4>Input:</h4> s = "00011000" <br><h4>Output:</h4> 2 <br><h4>Explanation:</h4> We flip to get 00000000. <br><h4>Constraints:</h4> 1 <= s.length <= 105 s[i] is either \'0\' or \'1\'.
