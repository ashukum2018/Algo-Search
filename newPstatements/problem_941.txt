You have n dice and each die has k faces numbered from 1 to k. Given three integers n, k, and target, return the number of possible ways (out of the kn total ways) to roll the dice so the sum of the face-up numbers equals target. Since the answer may be too large, return it modulo 109 + 7. <br><h4>Example 1:</h4> <br><h4>Input:</h4> n = 1, k = 6, target = 3 <br><h4>Output:</h4> 1 <br><h4>Explanation:</h4> You throw one die with 6 faces. There is only one way to get a sum of 3. <br><h4>Example 2:</h4> <br><h4>Input:</h4> n = 2, k = 6, target = 7 <br><h4>Output:</h4> 6 <br><h4>Explanation:</h4> You throw two dice, each with 6 faces. There are 6 ways to get a sum of 7:</h4> 1+6, 2+5, 3+4, 4+3, 5+2, 6+1. <br><h4>Example 3:</h4> <br><h4>Input:</h4> n = 30, k = 30, target = 500 <br><h4>Output:</h4> 222616187 <br><h4>Explanation:</h4> The answer must be returned modulo 109 + 7. <br><h4>Constraints:</h4> 1 <= n, k <= 30 1 <= target <= 1000
