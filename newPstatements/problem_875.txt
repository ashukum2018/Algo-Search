Given an array points where points[i] = [xi, yi] represents a point on the X-Y plane, return true if these points are a boomerang. A boomerang is a set of three points that are all distinct and not in a straight line. <br><h4>Example 1:</h4> <br><h4>Input:</h4> points = [[1,1],[2,3],[3,2]] <br><h4>Output:</h4> true <br><h4>Example 2:</h4> <br><h4>Input:</h4> points = [[1,1],[2,2],[3,3]] <br><h4>Output:</h4> false <br><h4>Constraints:</h4> points.length == 3 points[i].length == 2 0 <= xi, yi <= 100
