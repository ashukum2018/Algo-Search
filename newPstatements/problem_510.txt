You are given an integer array nums consisting of n elements, and an integer k. Find a contiguous subarray whose length is equal to k that has the maximum average value and return this value. Any answer with a calculation error less than 10-5 will be accepted. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [1,12,-5,-6,50,3], k = 4 <br><h4>Output:</h4> 12.75000 <br><h4>Explanation:</h4> Maximum average is (12 - 5 - 6 + 50) / 4 = 51 / 4 = 12.75 <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [5], k = 1 <br><h4>Output:</h4> 5.00000 <br><h4>Constraints:</h4> n == nums.length 1 <= k <= n <= 105 -104 <= nums[i] <= 104
