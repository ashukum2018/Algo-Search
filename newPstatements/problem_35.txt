Given an array of integers nums sorted in non-decreasing order, find the starting and ending position of a given target value. If target is not found in the array, return [-1, -1]. You must write an algorithm with O(log n) runtime complexity. <br><h4>Example 1:</h4> <br><h4>Input:</h4> nums = [5,7,7,8,8,10], target = 8 <br><h4>Output:</h4> [3,4] <br><h4>Example 2:</h4> <br><h4>Input:</h4> nums = [5,7,7,8,8,10], target = 6 <br><h4>Output:</h4> [-1,-1] <br><h4>Example 3:</h4> <br><h4>Input:</h4> nums = [], target = 0 <br><h4>Output:</h4> [-1,-1] <br><h4>Constraints:</h4> 0 <= nums.length <= 105 -109 <= nums[i] <= 109 nums is a non-decreasing array. -109 <= target <= 109
