The array-form of an integer num is an array representing its digits in left to right order. For example, for num = 1321, the array form is [1,3,2,1]. Given num, the array-form of an integer, and an integer k, return the array-form of the integer num + k. <br><h4>Example 1:</h4> <br><h4>Input:</h4> num = [1,2,0,0], k = 34 <br><h4>Output:</h4> [1,2,3,4] <br><h4>Explanation:</h4> 1200 + 34 = 1234 <br><h4>Example 2:</h4> <br><h4>Input:</h4> num = [2,7,4], k = 181 <br><h4>Output:</h4> [4,5,5] <br><h4>Explanation:</h4> 274 + 181 = 455 <br><h4>Example 3:</h4> <br><h4>Input:</h4> num = [2,1,5], k = 806 <br><h4>Output:</h4> [1,0,2,1] <br><h4>Explanation:</h4> 215 + 806 = 1021 <br><h4>Constraints:</h4> 1 <= num.length <= 104 0 <= num[i] <= 9 num does not contain any leading zeros except for the zero itself. 1 <= k <= 104
